# Q_links - Быстрые ссылки на карты

Расширение для браузеров (Chrome, Edge, Opera, Brave и другие на базе Chromium), позволяющее быстро открывать текущие координаты на различных картографических сервисах.

## Описание

Q_links - это удобное расширение для браузера, которое извлекает координаты из текущей вкладки и позволяет одним кликом открыть эту же локацию на других картографических сервисах.

## Возможности

### Поддерживаемые сервисы:

- **NSPD** - Национальная система пространственных данных России
- **CAD.by** - Кадастровая карта Беларуси
- **2GIS** - с поддержкой регионов:
  - Россия (основная кнопка)
  - UZ - Узбекистан
  - KZ - Казахстан
  - TJ - Таджикистан
  - BY - Беларусь
  - AZ - Азербайджан
  - AM - Армения
- **GoogleMap** - Google Maps
- **NAKARTE** - nakarte.me
- **OSM** - OpenStreetMap
- **BingMap** - Bing Maps
- **HereMap** - Here Maps
- **Wikimapia**
- **Mapillary**
- **SC** - Speedcam Online
- **RB** - RadarBase
- **MC** - Mapcam
- **Retromap**
- **Waze**
- **Pastvu**
- **PKK_new** - Публичная кадастровая карта России

## Установка

### Из исходного кода:

1. Скачайте или клонируйте репозиторий:
```bash
git clone https://github.com/MOneK292/Q_links.git
```

2. Откройте браузер и перейдите на страницу расширений:
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
   - Opera: `opera://extensions/`
   - Brave: `brave://extensions/`

3. Включите "Режим разработчика" в правом верхнем углу

4. Нажмите "Загрузить распакованное расширение"

5. Выберите папку с расширением

## Использование

1. Откройте любой картографический сервис с координатами в URL
2. Кликните на иконку расширения Q_links
3. Выберите нужный сервис из списка
4. Новая вкладка откроется с теми же координатами на выбранном сервисе

### Дополнительные возможности:

- **Левый клик** - открыть в новой активной вкладке
- **Средний клик (колесико мыши)** - открыть в фоновой вкладке

## Технические детали

Расширение извлекает координаты из URL текущей вкладки и преобразует их в формат, подходящий для каждого картографического сервиса. Поддерживаются различные системы координат и проекции.

Использует Manifest V3 для совместимости с современными браузерами.
